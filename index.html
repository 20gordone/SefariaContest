Welcome to the Sefaria Acrostic applet! This was designed for the Sefaria API Contest in the summer of 2020. It uses the Sefaria API to search Tanakh (the Old Testament).

The applet's purpose is to find biblical verses that are acrostics for hebrew names. These verses can be used to make personalized Jewish art. For example, the default name to search for is "Leah" and "Benjamin" in hebrew. A piece of wedding artwork for "Leah" and "Ben" is shown to the right.

To use the applet, write in the name/names in hebrew in the textbox without spaces. If you don't have a hebrew keyboad, you can go to this website. Then choose which books you want to search. Once you've chosen your names and books, click "Find Verses". The results will appear in a dropdown, and you can mouse over the options to see the full English translation of each result. For any of the results, you can render what the acrostic art could look like. You may need to scroll down to see the entire acrostic.
